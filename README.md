# 项目介绍
本项目为个人博客网站，采用Zensical打包+Github托管，以域名benobo.me呈现，依照仓库Wcowin/Zensical-Chinese-Tutorial相关教程搭建，用来记录个人学习过程，技术分享，书籍分享，生活随笔。核心目的是搭建自我表达的个人空间，利用互联网可视化空间展出。
# 仓库描述与搭建过程
## 基本架构与操作流程：
本地编辑markdown文件，利用zensical工具转化为HTML网页文件，托管至GitHub，Actions负责读取docs文件，依照zentical.toml文件调用云端服务器，进行相关操作，将结果展示至Github Pages,并加入个人域名便于访问。
## 具体搭建过程：
配置Zensical+连接至GitHub Pages+个人域名包装。更多具体操作详见前辈仓库Wcowin/Zensical-Chinese-Tutorial。
# 后续操作
逐步完善网页结构+内容输出。内容输出无固定主题，较为发散，暂定为个人生活表达+知识分享。仍在探索，敬请期待，欢迎交流。
